# Options

# tag

  • Type: string
  • Default -

Suffix to append to the Jsonic instance identifier. Useful if you have to debug multiple instances with different options and plugins.

# Example

let jsonic = Jsonic.make({tag:'foo'})
jsonic.id // === 'Jsonic/1614085851902/375149/foo'
